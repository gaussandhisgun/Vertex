# Vertex
A simple, minimalistic and slightly colorful theme for `xfce4` (commonly referred to as "rat desktop" or just "rat"). Still has some bugs but good enough to work.

# Installation
```bash
cd ~/.themes
git clone https://github.com/gaussandhisgun/Vertex.git
```
Select the theme in `Window Manager` section of settings.
